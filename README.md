# geetest2-npm

Geetest 2.0 for NPM

NOTE: geetest2 does not require any 3rd library

## install

```sh
npm install geetest2 --save
```

## use

```js
var Geetest = require('geetest2');

// now can use Geetest constructor
// for more detail, refer the demo (dir demo/)
```

## run demo

```sh
npm install
npm start
// access http://localhost:9991
```
